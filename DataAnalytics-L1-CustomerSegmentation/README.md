# Customer Segmentation Analysis

## Project Overview
This project applies Customer Segmentation using the K-Means Clustering algorithm to group customers based on their purchasing behavior. The analysis helps businesses understand different customer segments and create targeted marketing strategies.

## Objective
- Analyze customer purchasing behavior
- Perform customer segmentation using K-Means Clustering
- Determine the optimal number of clusters using the Elbow Method
- Visualize customer segments
- Provide business recommendations based on the analysis
